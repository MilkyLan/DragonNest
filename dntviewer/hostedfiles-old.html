Equipment

itemtable_equipment
  • Equipment names made up of fields in NameIDParam.
  • EnchantID can be used to look up the item in enchanttable.
  • SellAmount has npc sell cost
  • LevelLimit required level to use the item
  • NeedJobClass set for items that only certain class can use
enchanttable
  • Use EnchantID from itemtable_equipment to find the equipment
  • The extra stats from enchantmetn are stored in the statevalue columns
  • Cost of the enhancement is stored in the NeedCoin column
  • Lustres, etc required are in the NeedItem columns - look up that id in itemtable to find the name of the item
